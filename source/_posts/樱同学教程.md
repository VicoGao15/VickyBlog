打开Visual Studio Code软件。
### 第一步：新建文章
```
hexo new <文章名称>
```
### 第二步：编辑文章
第一步执行完成后，会在source/_posts文件夹下生成同名md格式文件，直接打开编辑。

### 第三步：部署文章
文章编辑完成后，执行如下两个命令，部署上去。
```
hexo clean
```
```
hexo deploy
```
